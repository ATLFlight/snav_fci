This is an example of using the loop option of the trajectory generation library wherein a closed trajectory is generated from a series of waypoints. The vehicle loops through the trajectory endlessly until the user stops the mission with [Ctrl-C]. It also demonstrates the use of the snav_fci::FlightControlInterface::preempt_current_action() function for asynchronously preempting the currently-executing action.
